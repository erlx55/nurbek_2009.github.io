В HTML существует множество тегов для работы с текстом. Вот основные из них: 
                                              1.основные теги для текста
  <{p}> - абзац(параграв)
  <{br}> - перенос строки
  <{hr}> - горизонтальная линия(тематический раздель)
  <{pre}> - преформатированный текст(сохроняет пробелы и переносы)
  <{code}> - код программы(моноширинный шрифт)
  <{var}> - переменная(курсив)
  <{kdb}> - вывод с клавиатуры(моношривнный)
  <{samp}> - вывод программы(моноширинный)
                                    2.семантические теги(логческие форматирования)
  <{strong}> - важный текст(жирный)
  <{em}> - акцентированный текст(курсив)
  <{mark}> - выделеный текст(подсвечен)
  <{small}> - мелкий текст
  <{del}> - удаленный текст(зачёркнутый) 
  <{ins}> - вставленный текст(падчёркнутый)
  <{sub}> - подстрочный текст(нижный индекс)
  <{sup}> - надстрочный текст(вехный индекс)
  <{abbr}> - абревиатура(с отрибутом title для расшифровки)
  <{cite}> - цитата или источник(курсив)
  <{g}> - короткая цитата(в кавычках)
  <{blockquote}> - блок цитаты(с отступам)
  <{time}> - дата/время(с атрибутом datetime)
  <{dfn}> - определение термина(курсив)
                                                        3.загаловки
  <{h1}> - заголовок 1-го уровня(самый важный)
  <{h2}> - зоголовок 2-го уровня
  <{h3}> - заголовок 3-го уровня
  <{h4}> - заголовок 4-го уровня
  <{h5}> - заголовок 5-го уровня
  <{h6}> - заголовок 6-го уровня(наименне важный)
                                                     4.теги для списков
  <{ul}> - маркированый список
  <{ol}> - нумерованный список
  <{li}> - элемент списка
  <{dl}> - список описаний
  <{dt}> - термин в списке описаний
  <{dd}> - описания термина
                                                   5.другие полезные теги
  <{span}> - контейнер для стилизации части текста
  <{div}> - блочный контейнер(сематичный)
  <{b}> - жирный текст(без симетичкой важности)
  <{i}> - курсив без сематического акцента
  <{u}> - подчёрнутый текст
  <{s}> - зачёркнутый текст
                                                        эти теги помаают структурировать и оформлять текст в html.
                                                       для стилизации лучше использовать css,но html предоставляет
                                                                 базовые возможности форматирования
